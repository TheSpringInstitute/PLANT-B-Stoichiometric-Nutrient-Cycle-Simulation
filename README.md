This is a repository by Livian Von Dran, one of the entomologists for the PLANT-B CubeSat Terrarium Mission. Livian is making this repository as part of their final project for their Professional Graduate Certificate in Earth Data Analytics program. 

This repository will support efforts to simulate stoichiometric nutrient cycles occurring inside the PLANT-B terrarium. This project will be divided into two parts: habitat mapping and nutrient cycle simulation. Utilizing spatiotemporal data is a requirement for the final project, so mapping will be performed to identify territories where the chosen invertebrate and plant species coexist in close proximity to one another. If such an area is identified, data such as soil nutrient concentrations and present soil microbes will be pulled to identify the optimal environmental conditions for the terrarium's organisms.   

Please feel free to use any data generated in this repository as needed. If you feel that changes need to be made to the repository to better suit the PLANT-B mission, email Livian at lvondran23@gmail.com with your suggestions or create a fork of this repository.
